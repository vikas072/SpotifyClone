## ✨ About

This is a **Spotify-inspired music UI** built using only **HTML and CSS**.

I created this project to practice frontend design, layout building, responsive styling, and modern UI cloning.  
Right now it is a static frontend project, but my future goal is to make it work like a real music streaming platform.

---

## 🚀 Features

- 🎧 Spotify-style dark interface  
- 🎵 Music cards and playlist layout  
- 📌 Sidebar navigation  
- 🎨 Clean and modern design  
- 📱 Responsive UI  
- ⚡ Built with pure HTML & CSS
