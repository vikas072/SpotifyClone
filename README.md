# 📖 About

This project is a **frontend clone of Spotify** designed to replicate the modern look and feel of the original platform using only **HTML5** and **CSS3**.

It focuses on creating a clean user interface, responsive layouts, attractive music cards, and a smooth user experience while strengthening frontend development skills.

