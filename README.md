 # 🎵 Spotify Clone

A responsive **Spotify Clone** built using **HTML and CSS**. This project recreates the look and feel of Spotify's user interface with a modern layout, sidebar navigation, music cards, and clean styling.

## 🚀 Features

- Responsive Spotify-style UI
- Sidebar navigation
- Music playlist/card layout
- Modern and clean design
- Built with pure HTML and CSS

## 🛠️ Tech Stack

- HTML5
- CSS3
